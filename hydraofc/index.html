The Hydra ordinal collapsing function

a notation that i made to give a scale to function like Veblen function and Bachmann's ordinal collapsing function when i was learning about them, which i then formalised into a function

definition

f1+N(0, n) = n
f1+N(1, n) = S(n)
f1+N(1+ωN+1g1+N(0)+a, n) = f1+N1+a(ωN+1g1+N(0), n) aka (f1+N(ωN+1g1+N(0), .))1+a(n)
f1+N(1+g1+N(ωN+1c+1(b+1)), n) = f1+N(g1+N(ωN+1c+1b+ωN+1c(ωN+n)), 0)
g1+N(x) is either x or any normal form of f1+N where at some point nm (where i >= m >= 1) = x and all nl (where m > l) = 0
ωN+1 > a
ωN+1 > b
ωN+2 > c
N >= 0
for f0 (N = -1), the definition is the same except the ordinal that would be ω-1 is replaced with a natural number (the base)

small and large hydra ordinal

the small hydra ordinal is h where
h is f1(f2(...fi(ωi, 1)..., 0), 0)
i is the omega fixed point a |-> ωa
the large hydra ordinal is h where
h is the fixed point h |-> f1(f2(...fi(ωi, 1)..., 0), 0)
i is the h-th omega fixed point a |-> ωa
normal form

the normal form for nonzero ordinals is f1+N(h1+f1+N(h2+...f1+N(hi+a, 0)+ni..., 0)+n2, 0)+n1 where
ωN > nm
ωN > a > 0
hm >= ωN+1
hm = ωN+1c1b1+ωN+1c2b2+...+ωN+1cjbj where
ωN > bm
ωN+2 > cm > cm+1 > 0
ω > j > 0
 N >= 0
ω > i
